# 🚀 Wealth IQ Runner

🎮 Where speed meets smart money decisions.

---

## 💡 Overview
Wealth IQ Runner is a unique gamified project that combines endless runner mechanics, IQ-based challenges, and financial & real estate literacy into one interactive experience.

Players progress by making smart decisions, solving quick challenges, and managing virtual assets such as cash, investments, and properties.

---

## 🎯 Concept
This project is designed to:
- Improve financial awareness
- Enhance decision-making skills
- Introduce real estate concepts
- Combine learning with fun gameplay

---

## 🎮 Game Mechanics

### 🏃 Runner Mode
- Collect cash and rewards
- Avoid bad financial decisions
- Choose between risky and safe paths

### 🧠 IQ Challenges
- Quick questions and logic puzzles
- Financial and real estate scenarios
- Reward smart answers with bonuses

### 🏠 Asset System
- Buy virtual properties
- Earn passive income
- Upgrade assets over time

---

## 💰 Learning Topics
- Cash flow vs expenses
- Smart investing
- Risk vs return
- Real estate basics
- Financial planning
